# E-Commerce Back End

## Description
```md
An application that configures an Express.js API and that uses Sequelize 
to interact with a MySql database for an E-Commerce site.
```

![Alt text](assets/images/13-orm-homework-demo-01.gif)

## Installation & Usage
```md
npm i
mysql -u root -p
SOURCE db/schema.sql;
exit mysql
npm run seed
npm start
```
